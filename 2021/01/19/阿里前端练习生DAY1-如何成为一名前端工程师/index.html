克军： 成为一名前端工程师

阿里云智能安全和大中台前端团队

前端发展历程和前景

  • “任何能用JS实现的应用最终都会有JS实现”

  • 技术的价值：界面的价值，物理界面最终都会向数字界面转化

  • 发展方向：一端到多端；GUI到NUI；B/S到云/端

​

大厂OR小厂？大厂能给前端工程师什么

  • ”丰富的业务使用场景和技术储备“

前端工程师画像：需要具备的技能

  • ”有没有刨根问底的好奇心“，动手实践能力，”会不会动手实现自己的想法“

  • 眼界决定了边界

怎样提高

  • 横向，多线程：

红色：工程实践；绿色：基础知识；蓝色：领域知识

  • 纵向：深入。学习不能停留于表面。

有效的学习方式

  • 问题解决之后还要总结和归纳，才能把这次实践转换成积累和经验

  • 如何从日常中发现问题：可以从追求代码质量开始，然后一步步从最求最优解的方案的方向提升自己

​ 比如，如图所示的思考题CSS就能实现，也就是说不要仅仅学一些工具层面的东西，他会限制你的思路，要（重视基础）从最基础的webapi学习，加深对技术的理解，才能创造性地解决问题。

团藏：前端技术应知应会

阿里UC事业部

前端发展史与体系概述

  • JS 在浏览器出现后几年出现，由DOM BOM和EcmaScript组成，ES是JS的核心，用于规范JS的语法结构，ES5，ES6(chrome/支持)

  • nodejs和npm把前端引入工程化时代；打包工具：wabpack。

  • MVVM框架 - angularsj最早

前端技术方向指引

互动技术
  • 夸克，webAR测试实例
  • Web3D引擎‘Oasis 3D’——蚂蚁金服自研的3的引擎，未对外开放，开发手机应用的小游戏
  • lottie-pixi动画设计实现，例如抽奖游戏、夸克AI身体测量，通过高性能的web run-time渲染

智能化：imgcook，将设计转换成代码
node.js
  • egg框架，基于KOA开发
    • serverless 无服务架构，免运维管理后端服务。前端同学不需要关心服务器的内容。
    • 阿里云实现函数计算服务

数据可视化

  • 大数据分析应用领域的最后一公里

  • 开箱即用的基础图表库、针对业务封装的可视化组件、针对多场景的可视化搭建平台

中后台
  • 解决搭建效率问题，提供统一规范组件标准，可视化搭建，提供开箱即用的SDK

  • 解决方案：AntDesiging pro

跨端技术

前端能力要求与学习途径

校招生
  • 需要对技术原理有一定深入的了解
  • 在前端领域的某一项技术上由深入研究
  • 技术上能做到从想法转换成行动

  • 参加过项目，有名的博主

npx create-react-app react-test-app --template typescript

  1. 安装react-redux和rap

npm i --save redux react-redux redux-devtools-extension rap

–save保存在package里

​ 2.X cnpm安装

npm install -g cnpm --registry=https://registry.npm.taobao.org

  1. 安装类型

cnpm i --save-dev @types/react-redux
